
# Compositionality of Sentences on IMDB & Amazon Reviews  
### Transformer-Based Embeddings + Classical Machine Learning

---

## Overview

This project performs large-scale sentiment analysis on two benchmark datasets:

-  **IMDB Movie Reviews**
-  **Amazon Product Reviews**

## Running the Project

Run notebooks in this order:
IMDB
IMDB_DataAnalysis.ipynb
imdb_code.ipynb

Amazon
AmazonReview_EDA.ipynb
amazon_reviews.ipynb
